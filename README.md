# List of openclaw alternatives

* [nanobot]([https://github.com/HKUDS/nanobot](https://github.com/HKUDS/nanobot)) [![nanobot github stars badge](https://img.shields.io/github/stars/HKUDS/nanobot?style=flat)](https://github.com/HKUDS/nanobot) - The Ultra-Lightweight alternative to OpenClaw

* [nanoclaw]([https://github.com/gavrielc/nanoclaw](https://github.com/gavrielc/nanoclaw)) [![nanoclaw github stars badge](https://img.shields.io/github/stars/gavrielc/nanoclaw?style=flat)](https://github.com/gavrielc/nanoclaw) - A lightweight alternative to OpenClaw that runs in Apple containers for security.
